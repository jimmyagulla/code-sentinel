# Glossary

| Term | Meaning |
|------|---------|
| Note | Structured annotation on a line range |
| Anchor | Line range + hash + snippet binding a note to code |
| Re-anchoring | Re-locating a note after the file changed |
| File health | OK / NEEDS_REVIEW / KO for a file |
| Reconcile | Reload storage, Git checks, anchor resolution |
